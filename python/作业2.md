## 第一题
打印所有 100 至 999 之间的水仙花数，所谓水仙花数是指各位数字立方和为该数字本身的整数。
```python
for num in range(100,1000):
    a = num // 100
    b = num % 100 // 10
    c = num % 10 
    if a**3 + b**3+ c**3 == num:
        print(num,end=',')
```

## 第三题
输出下三角九九乘法表。因为乘积位数不同，数据可能无法对齐。思考如何对齐数据。
```python
# 乘法表
for num1 in range(1,10):
```
